# Dieta do Zen PRO

Versão mais bonita e profissional, pronta para subir no GitHub e gerar APK por GitHub Actions.

## Como gerar o APK
1. Extraia o ZIP.
2. Suba tudo para um repositório no GitHub.
3. Abra a aba Actions.
4. Rode o workflow **Build Android APK PRO**.
5. Baixe o APK em Artifacts.

- visual premium escuro com verde fitness
- dashboard melhorado
- gráfico offline simples
- banco de alimentos
- meta diária
